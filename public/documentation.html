myMovies API This API provides information on movies, directors and genres. It also allows a user to register and create a list of favorite movies. Description: Get a list of all Movies URL: /movies Method: GET Query Parameters: None Request Body Data Format: None Response Body Data Format: JSON object containing data about all the Movies Description: Get data about a single movie based on the movie title URL: /movies/[Title] Method: GET Query Parameters: title, the title of the movie for which to obtain data Request Body Data Format: None Response Body Data Format: JSON object containing data about the specified movie, containing a title, description, genre, director, image URL and if-featured properties. Example: { title : "The Godfather", description : "American crime film about the Mafia based on novel by Mario Puzo.", genre : "crime", director : "Francis Ford Coppola", imageUrl : "https://www.rottentomatoes.com/m/godfather", ifFeatured : "yes" } Description: Get movie data about a genre by title/genre URL: /movies/list/:Genre Method: GET Query Parameters: genre, the genre of movies for which to obtain data Request Body Data Format: None Response Body Data Format: JSON object containing data about the specified movie, containing a title, description, genre, director, image URL and if-featured properties. Example: { title : "The Godfather", description : "American crime film about the Mafia based on novel by Mario Puzo.", genre : "crime", director : "Francis Ford Coppola", imageUrl : "https://www.rottentomatoes.com/m/godfather", ifFeatured : "yes" } { title : "The Shawshank Redemption", description : "American drama film about a banker sentenced to life in Shawshank State Prison.", director : "Frank Darabont", genre : "drama", imageUrl : "https://resizing.flixster.com/k6HGNQn7mJviYejJCX50YuYsYSo=/206x305/v1.bTsxMTE2NjcyNztqOzE4MTI5OzEyMDA7ODAwOzEyMDA", ifFeatured : "yes" } Description: Get data about a director by name URL: /directors/[Name] Method: GET Query Parameters: name, the name of the director for whom to obtain data Request Body Data Format: None Response Body Data Format: JSON object containing data about the specified director, containing a name, biography, birth year, and death year properties. Example: { name: "Francis Ford Coppola", biography: "Director of The Godfather.", birthYear: 1939, deathYear: null} } Description: Add a new user URL: /users Method: POST Query Parameters: None Request Body Data Format: JSON object holding data about the user to add with username, password, email, date of birth, and list of favorite movies Example: { "username" : "sarahb831", "password" : "notShown123", "email" : "smbeauchamp885@gmail.com", "dateOfBirth" : "15/03/2018", "favoriteMovies" : { "movie": "The Notebook" } } Response Body Data Format: JSON object containing data about the user that was added Example: Example: { "username" : "sarahb831", "password" : "notShown123", "email" : "smbeauchamp885@gmail.com", "dateOfBirth" : "15/03/2018", "favoriteMovies" : { "movie": "The Notebook" } } Description: Update a user's information URL: /users/[Username]/[Password] Method: PUT Query Parameters: username, the current username for this user password, the current password for this user Request Body Data Format: JSON object holding data about the user to update with username, password, email, date of birth Example: { "username" : "sarahbeauchamp", "password" : "notShown123", "email" : "smbeauchamp885@gmail.com", "dateOfBirth" : "15/03/2018", "favoriteMovies" : { } } Response Body Data Format: JSON object showing the updated information Example: { "username" : "sarahbeauchamp", "password" : "notShown123", "email" : "smbeauchamp885@gmail.com", "dateOfBirth" : "15/03/2018", "favoriteMovies" : { "movie": "The Notebook" } } Description: Add a movie to a user's list of favorites by username and movie title URL: /users/[Username]/[MovieTitle] Method: POST Query Parameters: username, the current username for this user movieTitle, the title of the movie to be added Request Body Data Format: None Response Body Data Format: JSON object showing the updated information Example: { "username" : "sarahbeauchamp", "password" : "notShown123", "email" : "smbeauchamp885@gmail.com", "dateOfBirth" : "15/03/2018", "favoriteMovies" : { "movie": "The Notebook", "movie": "John Wick" } } Description: Delete a movie from a user's list of favorites by username and movie title URL: /users/[Username]/[MovieTitle] Method: DELETE Query Parameters: username, the current username for this user movieTitle, the title of the movie to be deleted Request Body Data Format: None Response Body Data Format: Text message indicating whether or not the movie was successfully deleted from the list of favorites. Description: Delete a user from the list of users by username URL: /users/[Username] Method: DELETE Query Parameters: username, username for the user being deleted Request Body Data Format: None Response Body Data Format: Text message indicating whether or not the user was successfully deleted from the users list.
